# 구해줘 동물 (unity)

   ## save animals
   * ### Scene은 총3개
   * ###### (1). 수달방 --> 멸종위기동물 이름 알아맞추기 게임을 통해 코인 얻기 혹은 틀릴 경우 코인 잃음.
   * ###### (2). 상점  --> 동물들이 먹는 음식을 구매할 수 있음.
   * ###### (3). 마을회관 --> 동물 랜덤배치 + 각 동물들의 주식 나타내줌.
